Bitcoin-Starknet Bridge Documentation

Complete technical documentation for developers integrating with the Bitcoin-Starknet bridge

Quick Start Guide

1

Set Up Your Environment

Install the required dependencies and configure your development environment to interact with the Bitcoin-Starknet bridge.

# Install bridge SDK
npm install @starknet-bridge/sdk
# Or with Yarn
yarn add @starknet-bridge/sdk
2

Connect to the Bridge

Initialize the bridge client with your node configuration and network settings.

import { BridgeClient } from '@starknet-bridge/sdk';

const client = new BridgeClient({
  network: 'mainnet',
  bitcoinNode: 'your-bitcoin-node-url',
  starknetNode: 'your-starknet-node-url'
});
3

Perform Your First Transfer

Use the bridge to transfer Bitcoin to Starknet and back with simple function calls.

// Transfer Bitcoin to Starknet
const result = await client.deposit(
  amount: 0.1, // BTC
  recipient: 'starknet-address'
);

// Transfer back to Bitcoin
const withdrawal = await client.withdraw(
  amount: 0.1, // BTC
  bitcoinAddress: 'bitcoin-address'
);

API Reference

POST /api/v1/deposit

Initiate a Bitcoin deposit to Starknet. This creates a transaction that locks Bitcoin on the Bitcoin chain and mints equivalent wBTC on Starknet.

Parameter Type Description
amount number Amount of Bitcoin to deposit (in BTC)
recipient string Starknet address to receive wBTC
callbackUrl string (optional) URL to receive confirmation webhook
// Example request
{
  "amount": 0.5,
  "recipient": "0x072d30f36bd2e90d...f3ca5a7c771641e8e7",
  "callbackUrl": "https://your-app.com/webhooks/deposit"
}
POST /api/v1/withdraw

Initiate a withdrawal from Starknet back to Bitcoin. This burns wBTC on Starknet and unlocks Bitcoin on the Bitcoin chain.

Parameter Type Description
amount number Amount of wBTC to withdraw (in BTC)
bitcoinAddress string Bitcoin address to receive unlocked Bitcoin
priority string (optional) Transaction priority: "low", "medium", or "high"
// Example request
{
  "amount": 0.3,
  "bitcoinAddress": "bc1qar0srrr7xfkvy5l643lydnw9re59gtzzwf5mdq",
  "priority": "medium"
}
