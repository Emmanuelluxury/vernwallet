POST /api/v1/wallets/connect
Connect a user's Starknet wallet to the bridge service. Returns a connection token for authenticated requests.
Parameters
starknet_address
string
The Starknet wallet address to connect
required
signature
string
Signature proving ownership of the wallet
required
network
string
Network to connect to (mainnet, testnet)
optional
Request
Response
